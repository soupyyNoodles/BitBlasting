Q2 Route Blocking for Reducing Forest Fire Spread

Wildfires often spread across forest regions through natural terrain corridors such as dry vegetation paths, wind-aligned clearings, and forest access routes. Authorities can construct temporary firebreaks or block certain routes to slow down the spread. However, constructing firebreaks is costly, and only a limited number of routes can be blocked.

In this assignment, you will determine which forest routes should be blocked in order to minimize the expected spread of a wildfire.

---

Fire Spread Model

The forest is modeled as a directed graph G = (V, E, p), where

(a) V = {1, 2, . . . , n} represents forest regions (clusters of trees),

(b) E ⊆ V × V represents possible fire-spread routes (directed source-destination pairs), and

(c) p : E → (0, 1] assigns a propagation probability to each route.

The set {u |(u, v) ∈ E} is called the set of out-neighbors of u.

Let A0 ⊆ V denote the set of initially burning regions.

The fire evolves in discrete time steps:

• If region u is burning at time t, it gets exactly one opportunity to ignite each currently unburned out-neighbor v.

• The ignition attempt succeeds independently with probability p(u,v).

• If successful, v starts burning at time t + 1.

• Each tree cluster (node) burns at most once.

• The process stops when no new regions ignite.

Let A∞ denote the set of burned regions at termination.

We are interested in the expected total number of burned regions when we block some routes:

σ(R) = E[|A∞|],

where R ⊆ E is the set of blocked routes and the fire spreads on the modified graph with blocked routes removed

GR = (V, E \ R, p).

---

Route Blocking Problem

Given a budget k, select a set R ⊆ E such that

• |R| = k, and

• σ(R) is minimized.

Since the process is stochastic, σ(R) must be estimated using simulation.

---

Alternate Spread Model

Let h be a positive integer, then the h-hop spread model is defined by an additional constraint to the original fire spread model. Let

d(A0, v) = min over u∈A0 of graph-shortest-path-distance(src = u, dst = v)

then no nodes outside the set

Vh,A0 = {v | d(A0, v) ≤ h}

can be on fire at any time.

Your code will be evaluated on both spread models.
